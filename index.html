About Me

I'm a Computer Engineer with a passion for web development. I specialize in Ruby/Ruby on Rails and React/Redux. In my free time, I love to play chess, use computational power to solve mathematical problems from projecteuler.net and spend time with my amazing family.

Skills:
  • Advanced understanding of the core features of the Ruby programming language.
  • Advanced understanding of JavaScript core language concepts — DOM manipulation, Fetch API, ES6+ features.
  • Working knowledge of React/Redux as well as experience with responsive design interface development.
  • Advanced understanding of the Ruby on Rails web framework.
  • Strong understanding of fundamental database concepts – relational (SQL) databases, NoSQL databases, ORMs/ODMs and Web storage APIs (front end).
  • Working knowledge of unit, integration, e2e and UI testing, with RSpec & Capybara
